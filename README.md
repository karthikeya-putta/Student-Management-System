# Student Management System (Flask + MySQL)

## Features
- Add Student
- Edit Student
- Delete Student
- Search Student
- Flash messages
- Bootstrap UI

## Tech Stack
- Flask
- MySQL
- HTML, CSS, Bootstrap
- Jinja2

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run app:
   python app.py

3. Open:
   http://127.0.0.1:5000/
